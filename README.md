# 📚 Repositórios de Projetos – Curso Técnico em Desenvolvimento de Sistemas (2026)

Este conjunto de repositórios reúne todos os projetos desenvolvidos por mim ao longo das aulas do curso técnico em **Desenvolvimento de Sistemas**, realizado no ano de **2026**, durante o **terceiro ano do ensino médio**.

## 🏫 Instituição
Os projetos foram criados como parte das atividades práticas da escola **ETE FMC**, com o objetivo de aplicar conceitos teóricos em situações reais de desenvolvimento.

## 💻 Sobre os Projetos
Os repositórios incluem exercícios, trabalhos e aplicações desenvolvidas em sala de aula, abordando diferentes áreas da programação, como:

- Lógica de programação
- Desenvolvimento web
- Banco de dados
- Programação orientada a objetos
- Versionamento de código com Git

Cada projeto representa uma etapa do aprendizado e evolução ao longo do curso.

## 🎯 Objetivo
O principal objetivo destes projetos é demonstrar o conhecimento adquirido durante a formação técnica, além de servir como portfólio para futuras oportunidades acadêmicas e profissionais.

## 🚀 Observação
Os projetos podem variar em complexidade, pois foram desenvolvidos conforme o avanço dos conteúdos durante o curso.
